how to make animations in HTML and CSS

the text describes a red square and a blue ball that are animated using CSS. the square is 100x100 pixels in size and smoothly grows to 125x125 pixels when the mouse hovers over it due to a transition. the ball is round, blue, and moves automatically back and forth horizontally using an animation that repeats over and over again. the animation is controlled by keyframes that move the ball from left to right and back again.
