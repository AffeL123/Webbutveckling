how to make animations in HTML and CSS

The code uses CSS to animate a square and a ball. The red square starts at 100x100 pixels and smoothly grows larger when the mouse hovers over it. The blue ball is a circle that moves automatically from left to right and back again using a repeating CSS animation with smooth motion.

How to make it (the code)

.cube { width: 100px;
height: 100px;
background-color: red;
transition: width 2s, height 2s, transform 2s;
}
.cube:hover {
width:125px;
height: 125px;
}


.ball {
width: 40px;
height: 40px;
background: blue;
border-radius: 50%;
position: absolute;
animation: moveBall 2s ease-in-out infinite alternate;
}

@keyframes moveBall {
from {
left: 0;
}
to { left: 360px; } }
